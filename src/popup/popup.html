Adding sites

In every input field below, you can input the domain name of a website which you would like to be blocked whilst NoDistractions is enabled. ex: "wikipedia.org", or "social.distractingsite.co.uk".
Changes are saved automatically.
Tip: You can add *. before a site to block all of it's subdomains. "www" subdomains of entries are blocked automatically.

Advanced Usage

If you have more advanced needs of blocking you can prefix an entry with "!" and then use a URL Match Pattern.
ex: "!*://reddit.com/r/kittens/*"
or "!*://twitter.com/home".
Please keep in mind that the extension will only ever block webpages, never resource requests etc.
